# 位逻辑运算符

1、 & ：与
都是1，与出来的结果才是1，否则都是0
比如：a = 1, b = 0, a&b = 0 ； a = 1, b = 1, a&b = 1

2、| ：或
只要有一个是1那就是1，否则都是0
比如：a = 1, b = 0, a|b = 1 ；a = 0, b = 0, a|b = 0

3、~ ：取反
比如：a = 1，~a = 254
二进制：0000_0001 取反 1111_1110

4、^ ：异或
只要两数相同，那就是0，否则就是1
比如：a = 1, b = 1, a^b = 0；a = 1, b = 0, a^b = 1


# 位移运算符
1、<< 左移位
结果乘2
比如：a = 2<<2, a的结果是8
a = -1<<2, a的结果是-4

2、>> 右移
结果除以2
比如：a = 2>>1，a的结果是1

3、>>> 无符号右移，跟 >> 一样，除以2
